## :rocket: Technologies

This project was developed with the following technologies:

- [Node.js](https://nodejs.org/en/)
- [React](https://reactjs.org)

### Setup

```bash
$ cd server
$ yarn install
$ yarn knex:migrate

$ cd web
$ yarn install
```

### Run the application

```bash
$ cd server 
$ yarn start

$ cd web
$ yarn start
```
